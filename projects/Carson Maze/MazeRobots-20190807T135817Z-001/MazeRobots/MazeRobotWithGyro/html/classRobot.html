Uses PID control to drive the robot at a certain speed.

◆ getGyroHeading()

double Robot::getGyroHeading ( )
inline

Returns the heading of the robot in degrees.

The heading is in the range [0, 360) relative measured to 'East', where a more positive angle means more counter-clockwise if the robot is viewed from above.
